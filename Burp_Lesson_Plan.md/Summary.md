# 🎯 The 4 Web Exploitation Concepts

# Concept 1: HTTP Request Interception

Educational Goal: Understand how browsers and servers communicate
Students learn: All web traffic can be seen and modified
Real-world impact: Man-in-the-middle attacks, data theft


# Concept 2: Price Manipulation

Educational Goal:** Learn why backend validation is critical
Students learn: Never trust client-side data
Real-world impact: E-commerce fraud, millions in losses
Example: Change $10 to $0.01 in checkout


# Concept 3: Cookie & Session Manipulation
Educational Goal:** Understand how websites remember users
Students learn: How authentication works, session hijacking
Real-world impact: Account takeovers, identity theft
Example: Steal and decode JWT tokens

# Concept 4: SQL Injection Basics
Educational Goal:** See how attackers manipulate databases

Students learn: Input sanitization is essential
Real-world impact: Data breaches (Equifax - 147M people)
Example: Login bypass with ' OR '1'='1'--
